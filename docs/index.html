Solvra

Agentic Reasoning System for Complex Problem Solving

About

Solvra is an intelligent reasoning system that solves complex, structured problems through modular, explainable reasoning. Built for the Ethos 2025 ML Challenge, it uses specialized solvers and symbolic computation to tackle optimization, spatial reasoning, logic, and sequence problems.

Example Problems

Optimization Problem

Problem: Maria needs to bake a cake (1 hour), decorate room (1.5 hours), and pick up friend from airport (30 min away). She can decorate while cake bakes. What's the optimal sequence?

Solution: Solvra identifies: Start cake ‚Üí Decorate during baking ‚Üí Pick up friend. Total time: 2.5 hours.

Spatial Reasoning

Problem: A 3√ó3√ó3 cube has painted faces. How many unit cubes have exactly 2 painted faces?

Solution: Edge cubes excluding corners = 12 cubes with exactly 2 faces painted.

Sequence Pattern

Problem: Find the next number: 2, 5, 10, 17, 26, ?

Solution: Pattern is n¬≤ + 1. Next: 6¬≤ + 1 = 37.
