## 🚀 How to Use

### Quick Start
```bash
# Install dependencies
pip install -r requirements.txt

# Run the system
cd src
python main.py
```

### Output
The system will:
1. ✅ Load and preprocess data
2. ✅ Train on examples
3. ✅ Generate test predictions
4. ✅ Create performance reports
5. ✅ Save reasoning traces
